- **Test Object**
Login Page (`/login`)

- **Test Goal**
Verify that login with invalid credentials is rejected and an error message is displayed.

- **Test Activities**
1. Access the `/login` page.
2. Enter `admin` in the "Username" input box.
3. Enter `wrongpassword` in the "Password" input box.

**Test Object**
Access control for your profile page (`/profile`)


**Test Goal**
Verify that unauthenticated access to a protected page is blocked and a login prompt is displayed.


**Test Activities**
1. Ensure you are not currently logged in.
2. Access the `/profile` page.
3. Check the page displaying the text `Please log in to access your profile.`.
