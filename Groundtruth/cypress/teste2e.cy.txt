Login Page (Homepage Login Entry and Form Elements)

 **Test Goal**

 **Test Activities**

2. Click the login entry element.
3. Assert that the username input field `#user` exists.
4. Assert that the password input field `#password` exists.
5. Assert that the login button `.z-0` is visible.

Login Functionality (Incorrect Credential Handling)

**Test Goal**

**Test Activities**
1. Access `http://localhost:3000`.
2. Click the login entry.
3. Enter `9999999` in the username input box.

5. Click the login button.
6. Assert the error message element `[data-content=""] > div` to be visible.

**Test Object**
Login and Role Management Functionality

**Test Goal**
Verify that after logging in with valid credentials, you can access the management interface and successfully create a new role.


**Test Activities**
1. Access `http://localhost:3000`.
2. Click the login entry.
3. Enter the username.
4. Enter the password.
5. Click the login button and wait 3 seconds.
6. Click the management menu link and wait 3 seconds.
7. Click the second sub-menu item and wait 3 seconds.
8. Click the "New" button and wait 3 seconds.
9. Enter the character name in the first input box.
10. Enter the description in the second input box and wait 3 seconds.
11. Click the "Save" button and wait 3 seconds.
